Este projeto é uma lista de tarefas criada com HTML, CSS e JavaScript, permitindo ao usuário adicionar e gerenciar atividades de forma prática.

A interface contém:

Um título destacando a lista de tarefas

Uma área onde as tarefas são exibidas dinamicamente (todoList)

Um campo de entrada para adicionar novas tarefas ao pressionar Enter

Interações controladas por app.js, que gerencia criação e exibição das tarefas

O objetivo é oferecer uma estrutura simples e funcional para manipular uma lista de afazeres no navegador.
